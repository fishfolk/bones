Expand description

AssetIo is a trait that is implemented for backends capable of loading all the games assets and returning the raw bytes stored in asset files.

Required Methods§

Source

fn enumerate_packs(&self) -> BoxedFuture<Result<Vec<String>>>

List the names of the non-core asset pack folders that are installed.

These names, are not necessarily the names of the pack, but the names of the folders that they are located in. These names can be used to load files from the pack in the load_file() method.

Source

fn load_file(&self, loc: AssetLocRef<'_>) -> BoxedFuture<Result<Vec<u8>>>

Get the binary contents of an asset.

The pack_folder is the name of a folder returned by enumerate_packs(), or None to refer to the core pack.
