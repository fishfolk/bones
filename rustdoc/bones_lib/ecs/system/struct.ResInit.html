Struct bones_lib::ecs::system::ResInit

pub struct ResInit<'a, T>(/* private fields */)
where
    T: HasSchema + FromWorld;
Expand description

SystemParam for getting read access to a resource and initialzing it if it doesn’t already exist.

Use Res if you don’t want to automatically initialize the resource.
